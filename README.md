如何安装？
---
* 下载最新版本插件。
* 将 LiteItemShow-0.x.x.jar 文件放入 plugins 文件夹中。
* 启动服务器。 (仅支持Paper及衍生服务端)
* 现在插件应该在 /plugins 命令中显示为绿色。 如果显示红色或不存在，请检查控制台日志的错误信息。

环境需求
---
* Java 21+
* Paper 1.20+
